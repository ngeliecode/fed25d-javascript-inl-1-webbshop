// Denna README är min kom ihåg-lista

'npm install' --> installerar dependencies (npm-paket) --> [node_modules] i root

'npm run dev' --> kör scripts i 'package.json' (startar vite) --> 'localhost'

'npm --save-dev sass' --> [package-lock.json] håller reda på all dependencies och "sass" lagts till i package.json

'npm run build' --> dist
